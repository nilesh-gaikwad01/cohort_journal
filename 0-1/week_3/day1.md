### Commands - 

- pwd - It showa your working directory
- cd - It used for change directory 
  - `cd..` go inside the folder 
  - `cd../..` one step back 
  - `cd 01/week1` go inside the nested folder
- ls - It usases list all current folders in the current directory `ls`
- mkdir - It create new Dir `mkdir`
- touch - It create new empty file `touch`
- cat - Print content of files `cat notes.txt`
  
- vi/vim - Open the filed into  vm editor It used for editing files without GUI
- mv - move or remane the files `mv notes.txt folder/`for move and `mv noes.txt newNotes.txt`
- cp - copy the file `cp a.txt b.txt` folders `cp -r folderA folderB`
  
- nvm - ` Node version Manager` usases to install multiple node version and swich them 
  - `nvm install 18`
  - `nvm use 18`
- Npm - ` Node Package Manager` usases to install npm packages
  - `npm init -y` create node project
  - `npm install express` install librarise
- Node Javascript Run Time enverirnment to learn the js files outside of the browser
  - `node app.js` run
